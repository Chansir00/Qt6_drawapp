# 这是一个用于绘画的Qt应用程序

## 简介:

本项目基于qtgraphicview架构，方便了图形项的选择、拖动等操作，并使用QSS进行渲染。



## 功能:

1. 创建图形，如圆，矩形，三角形，线等。

2. 支持编辑单个图形或多个图形，包括放大或缩小，路由，组合或分解等。

3. 支持将图像保存为txt文件，创建新文件，打开文件，保存文件。

4. 支持撤销、重做、全选、删除等操作

5. 支持改变图形的颜色
